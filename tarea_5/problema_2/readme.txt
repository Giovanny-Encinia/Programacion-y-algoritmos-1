Se debe de ejecutar el programa de manera estandar en codeblocks.
Se ha separado el codigo en distintas carpetas, el archivo main manda
a llamar a las funciones detro del archivo moving_3_3.c y el archivo
dentro de la carpeta metricas estadisticas donde contienen
las funciones para calcular el promedio y la mediana.

Para la entrada de datos el programa lee un archivo .txt
en donde esta almacenada la matriz, la primera linea debe de contener
el numero de filas y columnas de la matriz a la que se le encontrara
el moving average/median. 

Una vez leido este archivo se imprime en pantalla el archivo original,
y las matrices generadas.
